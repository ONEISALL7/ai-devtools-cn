# Roadmap

本路线图用于规划项目维护优先级，具体实现会根据 issue、PR 和社区反馈调整。

## 0.1.0 - 项目基础

- 建立 README、LICENSE、CONTRIBUTING、SECURITY、MAINTAINERS
- 添加 GitHub issue 和 PR 模板
- 添加基础 Markdown lint 工作流
- 发布第一批 AI 工程模板
- 编写项目定位和贡献标准

## 0.2.0 - 核心模板库

- 增加 PR review 模板
- 增加 issue triage 模板
- 增加测试生成模板
- 增加文档维护模板
- 增加 release note 模板
- 增加工具评估模板

## 0.3.0 - 示例和案例

- 添加真实项目 PR review 示例
- 添加 issue triage 示例
- 添加文档更新示例
- 添加 CI 排错示例
- 添加团队导入 AI 工具的流程示例

## 0.4.0 - 维护工作流

- 增加 label 规范
- 增加 release checklist
- 增加 maintainer weekly checklist
- 增加模板质量评分标准
- 增加过期内容审计机制

## 1.0.0 - 稳定版本

- 核心模板达到稳定状态
- 文档结构清晰
- 有持续 issue/PR 维护记录
- 有明确版本和 changelog
- 有来自真实用户的反馈或案例
