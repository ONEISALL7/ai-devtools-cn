# Maintainer Weekly Checklist

每周维护检查清单，用于保持开源项目健康。

## Issue

- [ ] 查看新 issue
- [ ] 给未分类 issue 添加 label
- [ ] 要求缺少复现信息的 issue 补充上下文
- [ ] 关闭重复或已解决 issue
- [ ] 把高频问题转成 FAQ 或文档任务

## Pull Request

- [ ] 查看新 PR
- [ ] 检查 CI 状态
- [ ] 初步判断是否需要 request changes
- [ ] 合并低风险文档修复
- [ ] 对停滞 PR 留下下一步建议

## Documentation

- [ ] 检查 README 是否仍准确
- [ ] 更新 changelog
- [ ] 标记过期内容
- [ ] 补充常见问题

## Release

- [ ] 检查是否需要 patch release
- [ ] 整理 release note 草稿
- [ ] 确认 breaking changes 是否有迁移说明

## Community

- [ ] 回复贡献者问题
- [ ] 感谢首次贡献者
- [ ] 标记适合新手处理的 issue
- [ ] 记录下一周维护重点
