## Feedback

使用的模板或功能：
`pr-review` - PR Review 模板，配合 `doctor` 和 `trial` 命令。

使用场景：
审查一个文档入口 PR，确认 README、CLI 文档和第一批用户试用计划是否一致。

项目类型：
公开开源文档/CLI 项目。

是否解决问题：
基本解决。试用包把 review 输入和反馈草稿放在同一个目录里，比单独复制模板更容易上手。

少整理了一次 review checklist，也更容易把 AI 输出转成 PR comment。
